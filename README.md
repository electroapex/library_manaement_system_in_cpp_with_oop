# 📚 Library Management System

A console-based application built in **C++** that handles the day-to-day operations of a library.
It allows librarians to manage books and members, issue and return books, and automatically calculate fines for overdue returns — all through a simple menu-driven interface.

Data is saved using **file handling**, so records are preserved between sessions without any database.

Built with **OOP principles** — classes, inheritance, and polymorphism — making the codebase clean, modular, and easy to extend.
